# REST API Conventions

```
  * GET     /items              ->  get collection
  * PUT     /items              ->  replace collection
  * POST    /items              ->  create new item
  * DELETE  /items              ->  delete collection
```
```  
  * GET     /items/:id          ->  get item
  * PUT     /items/:id          ->  create/replace item
  * DELETE  /items/:id          ->  delete item
```
